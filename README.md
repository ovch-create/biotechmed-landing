# БиотехМед — лендинг интенсива

## Описание

Лендинг для профессионального интенсива **БиотехМед** от бренда-производителя **ANAGRAN**. Мероприятие направлено на врачей и нутрициологов, которые хотят системно разобраться в нутрицевтике и биотехнологиях на основе доказательной базы.

Дата: 23 мая 2026 года, Санкт-Петербург.

## Особенности

- Современный тёмно-синий дизайн с зелёным акцентом
- Адаптивная вёрстка для мобильных, планшетов и десктопа
- Все контентные данные вынесены в отдельные JS-файлы
- Квадратные плейсхолдеры вместо фото спикеров (для замены)
- Динамический рендер карточек (спикеры, сегменты, тарифы, отзывы) через JavaScript
- Кнопка «Наверх» с плавной анимацией

## Структура проекта

```
biotechmed-landing/
├── index.html          — основной HTML-файл
├── css/
│   └── style.css       — все стили, переменные, адаптивность
├── js/
│   └── main.js         — рендер динамических секций
├── data/
│   ├── speakers.js     — данные 6 спикеров
│   ├── segments.js     — 6 сегментов целевой аудитории
│   ├── reviews.js      — 5 отзывов участников
│   └── pricing.js      — 3 тарифа участия
└── README.md
```

## Секции лендинга

1. **Navbar** — фиксированная навигация с кнопкой регистрации
2. **Hero** — заголовок, описание, карточка с деталями события
3. **Проблема** — 3 болевые точки целевой аудитории
4. **О мероприятии** — описание формата, темы, статистика
5. **Для кого** — 6 сегментов B2B-аудитории
6. **Спикеры** — карточки с плейсхолдерами фото, именем и темой доклада
7. **Что получите** — 6 выгод от участия
8. **Форматы участия** — 3 тарифа с ценами
9. **Отзывы** — цитаты участников предыдущих интенсивов
10. **CTA-блок** — финальный призыв к регистрации
11. **Footer** — контакты, Telegram, сайт

## Как вставить фото спикеров

Откройте `data/speakers.js` и добавьте поле `photo` с путём к изображению. В `js/main.js` замените плейсхолдер на тег `<img>`.
